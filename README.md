# YOLO11---Solanum-betaceum-Cav-

Detección de la madurez del fruto de *Solanum betaceum* Cav mediante aprendizaje profundo basado en YOLO11.

## Archivos del repositorio

Este repositorio contiene:

- Notebook de clasificación y detección de madurez del fruto.
- Figuras del proyecto.
- Resultados obtenidos en formato Excel.

## Dataset completo

El archivo pesado del proyecto se encuentra publicado en Zenodo:

**DOI:** 
https://doi.org/10.5281/zenodo.21167174

## Repositorio

Este repositorio está asociado al dataset publicado en Zenodo para la detección de madurez del fruto de *Solanum betaceum* Cav usando YOLO11.
